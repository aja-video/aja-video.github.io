Detailed Description

This object is embedded in the AUTOCIRCULATE_TRANSFER struct that's returned from the CNTV2Card::AutoCirculateTransfer function, and contains status information about the transfer and the state of AutoCirculate.

Note
This struct uses a constructor to properly initialize itself. Do not use memset or bzero to initialize or "clear" it.
