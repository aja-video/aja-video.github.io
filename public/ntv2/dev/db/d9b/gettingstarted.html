Getting Started

Downloading and Installing the SDK

Developers no longer need to be registered with AJA Video Systems in order to download and use the NTV2 SDK. Starting with SDK 16.0, it is open-source under the MIT license, and is publicly available at the AJA GitHub repository.

Please refer to the README.md document in the repository for further instructions.

Accessing Proprietary Content

Registered AJA OEM Developer Partners have access to AJA proprietary content, including full documentation, special software, Knowledgebase articles, pre-release SDKs, tools and firmware, historic SDKs, firmware for current and end-of-life’d devices, plus access to first-class SDK technical support.

To access this content, log into the SDK support site using the credentials provided by AJA.

The SDK Support home page’s upper-right corner has two clickable icons:

  • Search (Magnifying Glass icon) — click to search the Knowledgebase pages
  • User (User icon) — click to pop up a menu:
    • Requests — Click to show all requests (support tickets)
    • Profile — Click to edit your personal profile (name, email address, password, language, time zone, etc.)
    • Log out — Logs out of the portal

The rest of the page is dedicated to making requests:
What can we help you with?

  • Development Questions — “How do I …?”
  • Development Troubleshooting — “Why isn’t XYZ working?”
  • Bug Report — “I think I found a bug in XYZ”
  • Feature Request — “I wish …”
  • Other — for all other requests that don’t fit the above categories
