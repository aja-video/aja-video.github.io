Detailed Description

Base class of objects that can serve device operation RPCs with NTV2RPCClientAPI instances.

  • The Create factory method attempts to find and load a plugin that instantiates a server with a requested configuration.

static NTV2RPCServerAPI* NTV2RPCServerAPI::CreateServer ( const std::string &  inURL)
static

Factory method that instantiates a new NTV2RPCServerAPI instance using a plugin based on the specified URL.

Parameters
[in]inURLSpecifies the server configuration. The URL scheme (required) determines the plugin to load. It must start with "ntv2", and the remainder of the name must match a DLL/dylib/so name.
Returns
If successful, a non-zero pointer to the new NTV2RPCServerAPI instance; otherwise nullptr (zero).
